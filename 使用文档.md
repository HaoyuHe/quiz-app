# 智能学习助手 - 使用文档

## 一、项目简介

智能学习助手是一个 Web 端的刷题/考试工具，支持从 Word/PDF 文件自动导入题库，提供练习、考试、错题本等功能。数据存储在后端服务器，所有设备访问同一份数据，实现多人共享。

### 核心特性

- 支持 Word（.docx）和 PDF 文件自动解析导入题目
- 支持单选题、多选题、判断题、填空题四种题型
- 多题库管理，支持题库重命名
- 抽查练习（不限时）和模拟考试（限时）
- 错题本自动收录，答对自动移除
- 学习记录和正确率统计
- 多设备数据共享（手机、平板、电脑）

## 二、环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Mac / Windows / Linux |
| Node.js | v18 或更高版本 |
| 网络 | 启动公网隧道时需要联网 |

### 检查 Node.js 是否已安装

打开终端（Mac）或 PowerShell（Windows），执行：

```bash
node --version
```

- 显示版本号（如 `v20.x.x`）→ 已安装，可以跳过
- 显示 `command not found` → 需要安装

### 安装 Node.js

访问 https://nodejs.org ，下载左侧 LTS 版本，双击安装即可。

## 四、启动服务

### 4.1 启动后端服务器

打开终端，进入项目目录并启动：

```bash
cd ~/Desktop/quiz-app
node server.js
```

启动成功后会显示：

```
=========================================
   智能学习助手 - 服务器已启动
=========================================

  本地访问:   http://localhost:3000
  网络访问:   http://192.168.x.x:3000
=========================================
```

> **这个终端窗口不要关闭**，最小化即可。

### 4.2 开启公网访问（可选）

如果需要让其他人在外网访问（不在同一 WiFi 也能打开），需要额外开一个隧道：

**新开一个终端窗口**（Mac 按 `Command + T`，Windows 打开新的 PowerShell），执行：

```bash
ssh -o StrictHostKeyChecking=no -R 80:localhost:3000 nokey@localhost.run
```

等待几秒，会显示类似：

```
xxx.lhr.life tunneled with tls termination, https://xxx.lhr.life
```

这个 `https://xxx.lhr.life` 就是公网地址，发给需要访问的人即可。

> **这个终端窗口也不要关闭**，最小化即可。

### 4.3 一键启动（Mac）

```bash
cd ~/Desktop/quiz-app
chmod +x start-online.sh
./start-online.sh
```

会同时启动服务器和公网隧道。

## 五、使用指南

### 5.1 访问地址

| 场景 | 地址 |
|------|------|
| 本机浏览器 | http://localhost:3000 |
| 同一 WiFi 的设备 | http://192.168.x.x:3000（启动时显示的地址） |
| 外网任意设备 | https://xxx.lhr.life（开启隧道后显示的地址） |

### 5.2 导入题库

1. 点击首页的 **"导入题库"**
2. 输入题库名称（如"新员工应知应会"），方便后续识别
3. 点击上传区域或拖拽文件，支持 `.docx` 和 `.pdf` 格式
4. 等待解析完成，预览题目数量和内容
5. 确认无误后点击 **"保存到题库"**

**支持的题目格式：**

- 题号：`1、` `1.` `1）` 等格式
- 选项：`A.选项内容` `B.选项内容`
- 答案：在题目末尾括号内，如 `（B）` `（ABC）` `（√）` `（×）` `（24）`
- 自动识别题型：单选、多选、判断、填空

### 5.3 抽查练习

1. 点击首页 **"抽查练习"**
2. 勾选要练习的题库（可多选）
3. 每个题库可单独筛选题型
4. 设置练习题数
5. 点击 **"开始练习"**

### 5.4 模拟考试

1. 点击首页 **"模拟考试"**
2. 勾选要考试的题库（可多选）
3. 设置考试题数和考试时间
4. 点击 **"开始考试"**
5. 倒计时结束自动交卷

### 5.5 错题本

- 答错的题目自动收录到错题本
- 可以针对错题进行专项练习
- 答对后自动从错题本移除

### 5.6 题库管理

1. 点击首页右上角 **"管理题库"**
2. 可以查看所有题库及题目数量
3. 点击编辑图标可以修改题库名称
4. 点击删除图标可以删除题库及其所有题目

### 5.7 数据管理

- **导出数据**：将所有题库、题目、错题、记录导出为 JSON 文件备份
- **导入数据**：从 JSON 文件恢复数据
- **清空数据**：删除所有数据（谨慎操作）

## 六、多设备共享说明

### 数据存储位置

数据存储在项目目录下的 `data/` 文件夹中（JSON 格式）：

```
quiz-app/
├── data/
│   ├── banks.json      # 题库信息
│   ├── questions.json   # 所有题目
│   ├── wrong.json       # 错题记录
│   └── records.json     # 学习记录
├── server.js            # 后端服务器
├── index.html           # 前端页面
└── ...
```

### 共享原理

```
设备A（电脑）──┐
               ├──→ 你的服务器（data/ 目录）──→ 所有设备看到同一份数据
设备B（手机）──┘
```

- 所有设备连接同一个服务器地址
- 数据统一存储在服务器的 `data/` 目录
- 任何人上传题目、做题，其他设备立即可见

## 七、常见问题

### Q: 公网地址每次启动都不一样怎么办？

`localhost.run` 免费隧道的地址是随机的。如果需要固定地址，可以：
- 注册 localhost.run 账号并绑定 SSH 密钥
- 或使用 Cloudflare Tunnel（需要注册 Cloudflare 账号）

### Q: 隧道断了怎么办？

重新执行 ssh 隧道命令即可，会生成新的公网地址。

### Q: 关闭电脑后服务还在吗？

不在。服务跑在你电脑上，电脑关机/休眠后服务就停了。重新开机后需要再次启动。

### Q: 如何备份数据？

两种方式：
1. **导出功能**：在页面上点"数据管理" → "导出数据"，下载 JSON 文件
2. **手动备份**：复制 `data/` 文件夹到其他位置

### Q: 换电脑后怎么迁移？

1. 在旧电脑上导出数据（JSON 文件）
2. 在新电脑上部署项目并启动
3. 在页面上点"数据管理" → "导入数据"，选择导出的 JSON 文件

### Q: 支持多少人同时使用？

没有人数限制，局域网或公网都可以访问。但建议仅限家人/小团队使用，因为服务器是轻量级的 Node.js 服务。

├── data/                # 数据存储（运行时生成）

## 九、技术栈

| 层级 | 技术 |
|------|------|
| 前端 | HTML + Tailwind CSS + 原生 JavaScript |
| 后端 | Node.js（内置 http 模块） |
