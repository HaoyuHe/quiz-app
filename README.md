# 智能学习助手 - 使用说明

## 快速开始

### 方式一：启动脚本（推荐）
- **Windows**: 双击 `start.bat`
- **Mac/Linux**: 双击 `start.sh` 或终端执行 `./start.sh`

然后在浏览器打开 http://localhost:8080

### 方式二：手动启动
在项目目录下执行：
```bash
python -m http.server 8080
```
然后浏览器打开 http://localhost:8080

---

## 功能说明

### 1. 导入题库
- 支持 `.docx`（Word）和 `.pdf` 格式
- 建议格式：
```
1. 题目内容
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：A
解析：答案解析内容
---
```

### 2. 练习模式
- **抽查练习**: 随机抽题，不限时
- **模拟考试**: 限时答题，倒计时提醒

### 3. 错题管理
- 答错自动归集到错题本
- 支持错题专项练习
- 答对后自动移除

---

## 数据存储

所有数据存储在浏览器本地（IndexedDB），不会上传到任何服务器。

### 数据迁移方法
如需在其他电脑使用，可：
1. 打开浏览器开发者工具（F12）
2. 切换到 Application → IndexedDB → QuizAppDB
3. 导出数据（或使用浏览器的数据同步功能）

---

## 离线使用

本版本已将所有依赖下载到本地 `libs/` 目录，支持完全离线使用。

## 常见问题

**Q: 为什么不能直接双击 index.html 打开？**
A: 浏览器安全策略限制了本地文件的 IndexedDB 和某些功能，需要通过 HTTP 服务器访问。

**Q: 换电脑后数据会丢失吗？**
A: 是的，数据存储在浏览器中。如需保留，请使用浏览器数据同步或手动导出。

**Q: 支持哪些浏览器？**
A: Chrome、Firefox、Safari、Edge 最新版本均可。
