## 🖥️ 电脑端使用

### Mac 系统

**方法一：双击启动（推荐）**
1. 双击 `start-mac.sh` 文件
2. 终端会自动打开并显示访问地址
3. 浏览器打开 http://localhost:8080

**方法二：终端启动**
```bash
cd quiz-app
./start-mac.sh
```

### Windows 系统

1. 双击 `start.bat` 文件
2. 浏览器打开 http://localhost:8080

## 📱 手机端使用

### 方式一：局域网访问（电脑需开着）

**步骤：**

1. **电脑端启动服务**
   - Mac: 双击 `start-mac.sh`
   - Windows: 双击 `start.bat`

2. **查看电脑 IP 地址**
   - 启动脚本会自动显示，如：`http://192.168.1.100:8080`
   - 或手动查看：
     - Mac: 系统偏好设置 → 网络 → 查看 IP
     - Windows: 命令行输入 `ipconfig`

3. **手机连接同一 WiFi**
   - 确保手机和电脑连接同一个 WiFi

4. **手机浏览器访问**
   - 输入 `http://电脑IP:8080`
   - 如：`http://192.168.1.100:8080`

5. **添加到桌面（可选）**
   - iOS Safari: 点击分享按钮 → 添加到主屏幕
   - Android Chrome: 点击菜单 → 添加到主屏幕

**优点**：简单，无需任何配置
**缺点**：电脑必须一直开着

---

### 方式二：GitHub Pages 部署（电脑可关闭）

**步骤：**

1. **创建 GitHub 仓库**
   - 登录 GitHub
   - 创建新仓库，如：`quiz-app`

2. **上传代码**
   ```bash
   cd quiz-app
   git init
   git add .
   git commit -m "初始化项目"
   git branch -M main
   git remote add origin https://github.com/你的用户名/quiz-app.git
   git push -u origin main
   ```

3. **开启 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 `main` 分支
   - 点击 Save

4. **访问网址**
   - 等待几分钟后访问：`https://你的用户名.github.io/quiz-app/`

5. **手机添加到桌面**
   - 手机浏览器打开网址
   - iOS: 分享 → 添加到主屏幕
   - Android: 菜单 → 添加到主屏幕
   - 之后可离线使用！

**优点**：电脑可关闭，手机随时访问，支持离线
**缺点**：需要 GitHub 账号，首次需联网

## ❓ 常见问题

### Q: 为什么不能直接双击 index.html 打开？
A: 浏览器安全策略限制了本地文件的某些功能，需要通过 HTTP 服务器访问。

### Q: Mac 提示"无法打开，因为它来自身份不明的开发者"？
A: 右键点击 `start-mac.sh` → 选择"打开" → 点击"打开"确认。

### Q: 手机无法访问？
A: 检查：
1. 手机和电脑是否连接同一 WiFi
2. 电脑防火墙是否阻止了 8080 端口
3. IP 地址是否正确

### Q: 数据会丢失吗？
A: 数据存储在浏览器本地（IndexedDB），清除浏览器数据会丢失。建议定期导出备份。

### Q: 换电脑后数据还在吗？
A: 不在。数据存在浏览器中，换电脑/换浏览器数据不互通。
